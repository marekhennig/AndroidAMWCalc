# AndroidAMWCalc

Android calculator application written in Java.

## Description

AndroidAMWCalc is a mobile calculator application for Android devices. The application provides basic arithmetic calculations with a user-friendly interface.

## Features

- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Clean and intuitive user interface
- Real-time calculation display

## Prerequisites

- Android SDK (API level 21 or higher)
- Java Development Kit (JDK 8 or later)
- Android Studio

## Installation

1. Clone the repository
```
git clone https://github.com/marekhennig/AndroidAMWCalc.git
```

2. Open the project in Android Studio

3. Connect an Android device or start an emulator

4. Build and run the application
```
./gradlew build
./gradlew installDebug
```

## Usage

1. Launch the application on your Android device
2. Enter numbers and select operations
3. Press the equals button to calculate the result
4. Clear button resets the calculator
